# BOLD APP UP — Démo GitHub Pages

Page HTML unique prête à publier.

## Déploiement rapide

1. Créer un nouveau repository GitHub public : `bold-app-up-demo`
2. Glisser-déposer les fichiers de ce dossier dans le repository
3. Aller dans **Settings > Pages**
4. Source : **Deploy from a branch**
5. Branch : **main** / folder : **root**
6. Attendre 1 minute
7. Ouvrir le lien GitHub Pages généré

## Fichiers

- `index.html` : page principale BOLD APP UP
- `.nojekyll` : évite les traitements inutiles de GitHub Pages

## Démo

Boutons à tester :

- Générer une cuisine
- Changer l’ambiance
- Générer le prédevis BOLD
- Exporter PDF
